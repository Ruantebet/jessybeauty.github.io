## 📝 Sobre o Projeto

Este projeto foi desenvolvido como uma solução de **Front-End puro**, focada em performance e experiência do usuário (UX). O objetivo é oferecer um catálogo elegante que reflita a identidade visual premium da marca (Dourado e Vermelho), permitindo a navegação por categorias e a finalização de pedidos via WhatsApp.

### ✨ Funcionalidades Principais
- [x] **Catálogo Dinâmico:** Renderização de produtos via JavaScript.
- [x] **Filtros por Categoria:** Navegação entre Maquiagem, Acessórios e Sexy Shop.
- [x] **Carrinho Local:** Itens salvos no navegador utilizando `localStorage`.
- [x] **Integração WhatsApp:** Conversão do carrinho em mensagem automática para fechamento de venda.
- [x] **Design Responsivo:** Adaptado para celulares e desktops.

---

## 🛠️ Tecnologias Utilizadas

Para este projeto, priorizei ferramentas modernas e eficientes de desenvolvimento:

* **HTML5 & CSS3:** Estrutura e semântica.
* **Tailwind CSS:** Estilização rápida e consistente com a paleta da logo.
* **JavaScript (ES6+):** Utilização de **Classes** para modelagem de produtos e **Módulos** para organização do código.
* **Vite.js:** Ferramenta de build para um ambiente de desenvolvimento ágil.
